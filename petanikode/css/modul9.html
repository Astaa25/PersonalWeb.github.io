Latihan CSS Border

Ini adalah contoh paragraf artikel buat Latihan properti border di CSS. Properti border adalah properti untuk membuat garis pada sisi elemen HTML. Garis ini berguna untuk memberikan batas antar elemen. Sehingga, bisa terlihat ada pemisah dan menjadi kesatuan.
Ini adalah contoh elemen dengan border image.
